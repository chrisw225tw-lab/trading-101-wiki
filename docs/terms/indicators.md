# 技術指標 (Technical Indicators) 📉

技術指標就像車子的儀表板。它不會開車 (那是你的工作)，但它會告訴你現在車速多快、油剩多少、引擎有沒有過熱。

## 1. 移動平均線 (Moving Averages, MA) 〰️

### 簡單移動平均 (SMA - Simple Moving Average)
*   **定義**: 過去 N 天收盤價的平均值。
*   **公式**: `(Day1 + Day2 + ... + DayN) / N`
*   **特色**: 反應比較慢 (Lagging)，但比較平滑，不容易被假動作騙。

### 指數移動平均 (EMA - Exponential Moving Average)
*   **定義**: 給越近的日期越大的權重 (Weight)。
*   **特色**: 反應比 SMA 快，對最新價格很敏感。短線交易者愛用。

### 關鍵均線 (The Key Lines)
*   **50 MA**: **中期趨勢**。機構法人 (大戶) 的防守線。
*   **200 MA**: **長期趨勢**。牛市與熊市的分界線。站上就是牛，跌破就是熊。
*   **黃金交叉 (Golden Cross)**: 短期線 (50) 向上穿過 長期線 (200) -> 大多頭訊號 🐂。
*   **死亡交叉 (Death Cross)**: 短期線 (50) 向下穿過 長期線 (200) -> 大空頭訊號 ☠️。

---

## 2. 相對強弱指標 (RSI - Relative Strength Index) 🌡️

*   **定義**: 比較一段時間內「漲幅」和「跌幅」的大小，來判斷多空力道。
*   **範圍**: 0 ~ 100。

### 怎麼用？
*   **Overbought (超買) > 70**: 
    *   買氣過熱，價格可能透支了。
    *   **Action**: 準備賣出，或不要追高。
*   **Oversold (超賣) < 30**: 
    *   恐慌拋售過頭，可能會有反彈。
    *   **Action**: 準備抄底。
*   **Divergence (背離)**:
    *   股價創新高，但 RSI 沒有創新高 -> **頂部背離** (可能要跌了)。
    *   股價創新低，但 RSI 沒有創新低 -> **底部背離** (可能要漲了)。

---

## 3. 指數平滑異同移動平均線 (MACD) 📊

*   **定義**: 用兩條不同速度的 EMA (快線與慢線) 的距離，來判斷動能 (Momentum)。
*   **組成**: 
    *   **MACD Line (快線)**
    *   **Signal Line (慢線)**
    *   **Histogram (柱狀圖)**: 兩條線的距離。

### 怎麼用？
*   **Crossover (交叉)**: 快線向上穿過慢線 -> **買進訊號**。
*   **Zero Line (零軸)**: 柱狀圖由負轉正 -> 動能轉強。

---

## 4. 布林通道 (Bollinger Bands) 🛤️

*   **定義**: 以 20 MA 為中心，上下各加減 2 個標準差 (Standard Deviation)。
*   **意義**: 95% 的價格走勢應該都在這個通道內。

### 怎麼用？
*   **Squeeze (擠壓)**: 通道變得很窄 -> 代表波動率極低，即將發生大行情 (暴漲或暴跌)。
*   **Mean Reversion (回歸)**: 價格碰到上軌通常會被打下來，碰到下軌通常會彈上去。

> **Buffett 的忠告**: 
> 指標只是輔助，不要過度依賴。**Price Action (價格行為)** 才是老大。如果指標說買，但價格在跌，請相信價格。
