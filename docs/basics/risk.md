# 風險管理 (Risk Management) 🛡️

這章最無聊，但也最重要。這是你唯一的保命符。

## 1. 止損 (Stop-loss)

**定義**：
在進場前就設定好的出場點。當價格觸及這個點位，代表你的 **Trade Idea** (交易想法) 已經 **Invalidated** (失效)，必須無條件離場。

**Buffett 的建議**：
*   永遠都要設 **Hard Stop** (硬止損)，不要只有心裡止損。
*   新手常見錯誤：虧損時 **Hold** (死抱) 期待反彈，最後小賠變大賠。

## 2. 倉位控管 (Position Sizing)

**定義**：
你這筆交易要買多少股/多少幣。

**黃金法則**：
單筆交易的風險 (Risk per trade) 不應超過總資金的 **1% - 2%**。

> **公式**：
> `買入股數 = (總資金 * 1%) / (進場價 - 止損價)`

## 3. 風險報酬比 (Risk-Reward Ratio, R:R)

**定義**：
你冒著 1 元的風險，是為了賺取幾元的利潤？

*   **建議**：至少要 **1:2** 或 **1:3**。
*   如果你冒險賠 $100，目標至少要賺 $200。這樣就算你勝率只有 50%，長期下來還是賺錢的 (Positive Expectancy)。

## 4. 回撤 (Drawdown)

**定義**：
資金從最高點 (Peak) 下跌到低谷 (Trough) 的幅度。

*   **Max Drawdown (最大回撤)**：你的帳戶歷史上最慘的一次跌幅。
*   **恢復難度**：跌 50% 需要漲 100% 才能回本。所以控制回撤比賺錢更重要。
