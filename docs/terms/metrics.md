# 股票關鍵指標 (Key Stock Metrics) 📊

當你打開看盤軟體 (如 Yahoo Finance, TradingView) 點開一支股票時，會看到一堆數字。這些是公司的 **Health Check** (健康檢查表)。

看懂這些，你才知道這家公司是 **Cheap** (便宜)、**Expensive** (貴)，還是 **Trash** (垃圾)。

## 1. 市值 (Market Cap) 💰

*   **定義**: `股價 x 總股數`。這家公司到底值多少錢。
*   **分類**:
    *   **Mega Cap (巨型股)**: > $200B (Apple, Nvidia)。通常最穩，波動小。
    *   **Large Cap (大型股)**: $10B - $200B。
    *   **Small Cap (小型股)**: $300M - $2B。波動大，成長空間大，但倒閉風險也高。
    *   **Penny Stock (雞蛋水餃股)**: < $5。通常是賭博，Buffett 建議新手遠離。

## 2. 本益比 (P/E Ratio) ⚖️

*   **定義**: **P**rice (股價) / **E**arnings (每股賺多少)。
*   **意義**: 你需要花多少錢，才能買到它賺的 1 塊錢利潤？
    *   **High P/E (高本益比)**: 例如 50x, 100x。代表市場預期它未來成長很快 (Growth Stock)，或者它真的很貴。
    *   **Low P/E (低本益比)**: 例如 10x, 15x。代表它可能被低估 (Value Stock)，或者是夕陽產業。
*   **注意**: 虧損的公司沒有 P/E (因為 E 是負的)。

## 3. 每股盈餘 (EPS) 💵

*   **定義**: **E**arnings **P**er **S**hare。公司賺的錢除以股數。
*   **意義**: 這是公司真實獲利能力的絕對數字。
    *   EPS 必須要 **Growing** (逐年成長)，股價才會長期上漲。
    *   **Earnings Surprise**: 如果公佈的 EPS 比分析師預期的好 -> 股價通常大漲。

## 4. 股價淨值比 (P/B Ratio) 📘

*   **定義**: **P**rice / **B**ook Value (公司淨資產)。
*   **意義**: 如果公司倒閉清算，把資產賣一賣，你手上的股票值多少錢？
    *   **P/B < 1**: 代表股價比公司淨資產還便宜！(通常是撿便宜的好機會，但也可能是公司有隱藏大雷)。

## 5. 殖利率 (Dividend Yield) 🎁

*   **定義**: 公司每年發給你的現金股利 / 股價。
*   **意義**: 存股族最愛。
    *   **High Yield**: 例如 5% 以上。通常是成熟公司 (電信、銀行、能源)。
    *   **No Dividend**: 成長型公司 (如 Tesla) 通常不發股利，因為他們把錢拿去再投資擴張。

## 6. 貝他係數 (Beta) ⚡

*   **定義**: 衡量這支股票相對於大盤 (S&P 500) 的波動程度。
    *   **Beta = 1**: 跟大盤連動。
    *   **Beta > 1**: 比大盤更活潑。Beta 1.5 代表大盤漲 1%，它通常漲 1.5%。(高風險高報酬)
    *   **Beta < 1**: 比大盤更穩。Beta 0.5 代表大盤跌 1%，它通常只跌 0.5%。(防禦型股票)

## 7. 成交量 (Volume) 🔊

*   **定義**: 今天這支股票成交了多少股。
*   **意義**: **Liquidity** (流動性) 的指標。
    *   **High Volume**: 很多人在買賣，進出容易，滑價小。
    *   **Low Volume**: 沒人氣，容易被操控，想賣賣不掉。
    *   **Volume Spike**: 成交量突然爆增 -> 通常代表有大事發生 (機構進場或逃命)，趨勢可能要反轉。
